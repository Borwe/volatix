# Volatix

An In memory database. Still in development <skill issues :)>

## Build the project

```bash
cargo build --release
```

## Start server

```
  cargo run --release --bin server
```

## Start cli repl

```
  cargo run --release --bin cli
```

Run some commands

```bash
SET name foo
GET name
DELETE name
HELP
```
